LES RACCOURCIS CLAVIER SOUS LINUX

Cette page traite des raccourcis clavier les plus fréquemment sollicités sous les utilitaires de Linux. Il ne s'agit évidemment pas d'une liste exaustive, en revange cette page vise à inciter les apprenants comme moi à les connaître étant donné leur intérêt dans le processus du codage. Le temps que ces outils permettent de gagner sera consacré et investi dans des manipulations plus élaborées.

DANS LE TERMINAL

Ctrl Shift + T Ajoute un nouvel onglet
Ctrl W Ferme l'onglet en cours
Ctrl C Ferme, annule le travail en cours
Ctrl Z Envoie le travail en cours en arrière-plan
Ctrl D Se deconnecter
Ctrl A Envoie le curseur en début de ligne
Ctrl E Envoie le curseur en fin de ligne
Ctrl U Efface la ligne
Ctrl K Efface la ligne depuis le curseur
Ctrl R + mot clé Recherche dans l'historique des commandes déjà tapées

Tab = Complétion automatique (des chemins ou des commandes) Flèche du haut ou du bas = Explore l'historique des commandes

DANS NAUTILUS

Ctrl H Afficher les fichiers cachés
Ctrl L Ouvrir un emplacement
Ctrl Alt+F1(ou F2, F3, F4, F5, F6) Affiche les différents terminaux
Ctrl Alt+F7 Affiche l'interface graphique
Ctrl Alt+L Verrouiller l'écran
Ctrl Alt+Suppr Déconnexion de la session
Ctrl alt+D Réduit toutes les fenêtres pour afficher le bureau
Alt F1 Affiche le menu Applications
Alt F2/td> Affiche la boite de dialogue "Lancer une application
Alt F9 Réduit la fenêtre actuelle
Alt F10 Augmente la fenêtre actuelle
Alt F5 Switch entre le mode normal et le mode plein écran d'une fenêtre
Alt F4 Ferme la fenêtre actuelle
Alt Entrée Lorsqu'un dossier est sélectionné, affiche ses propriétés
Ctl Alt Redémarre le serveur X
Alt Tab Switch entre les différentes fenêtres

DANS LES DOCUMENTS, FIREFOX, NAUTILUS

Ctrl Shift+N Créer un nouveau dossier
Ctrl A Sélectionner tout
Ctrl C Copier
Ctrl X Couper
Ctrl V Coller
Ctrl O Ouvrir un nouveau document
Ctrl N Créer un nouveau document
Ctrl S Sauvegarder un document
Ctrl W Fermer un document
Ctrl Q Fermer une application

EXEMPLE VISUEL

Quand on aime voyager, tout chemin mène à Rome, si l'on respecter le dead-line, les raccorcis vaut mieux implémenter plusieurs raccourcis.
